# AWS DevOps Jenkins Docker Kubernetes Interview Questions

A comprehensive collection of interview questions covering AWS, DevOps, Jenkins, Docker, and Kubernetes technologies.

## Topics Covered

- **AWS**: EC2, S3, Lambda, RDS, VPC, IAM, CloudFormation
- **DevOps**: CI/CD pipelines, Infrastructure as Code, monitoring, logging
- **Jenkins**: Job configuration, pipelines, plugins, distributed builds
- **Docker**: Containerization, Dockerfile, image management, networking
- **Kubernetes**: Pods, Services, Deployments, ConfigMaps, StatefulSets

## How to Use

1. Clone this repository
2. Navigate through each section
3. Review questions and answers
4. Practice answering before looking at solutions
5. Test your knowledge with real-world scenarios
